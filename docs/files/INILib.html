Package: SierraKomodo\INIController
Primary INI library class

Leveraging a provided SplFileObject pointing to an INI file, this class provides in-memory reading and modifying of data stored in INI files. Methods are also provided to write any changes made in memory to the INI file.

setKey(string $parSection, string $parKey, string $parValue) : void

Sets a key=value pair within an INI section header in memory.

Data passed into parameters is validated to ensure generated INI files will be properly formatted and will not produce any parsing errors.

Parameters are also trimmed of leading and trailing whitespace prior to validation for INI formatting purposes (I.e., saving '[section]' instead of '[ section ]', or 'key=value' instead of 'key = value'. This allows for better consistency in writing and reading of INI files between this class, parseini* functions, and any other programs written in other languages that may need to access these files.
